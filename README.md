# 语音和背景音乐分离算法及系统 python1190

## 项目概述

本项目是基于Python开发的语音和背景音乐分离算法及系统。系统功能完善，支持在线音频上传，并利用计算机神经网络算法实现对音频中人声与背景音乐的分离。分离后，可生成独立的背景音乐和人声两个新音频文件。

## 技术栈

- **后端语言**：Python 3.7
- **数据库**：MySQL
- **编译器**：PyCharm
- **前端技术**：未明确说明

## 功能模块

- 在线音频上传
- 音频人声与背景音乐分离处理
- 生成分离后的背景音乐与人声独立音频

## 系统角色

系统主要包括以下角色：

- 用户：上传音频，进行分离操作
- 系统：处理音频分离请求，生成分离后的音频

## 运行环境
